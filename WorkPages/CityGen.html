City Generation

  • Engine: Unity
  • Language: C#

This program generates a city by creating a grid that is split up into sections, then split up again into there own neighborhoods. The way I programmed it, somewhat works off the Binary Space Partitioning (BSP for short). 

While the program is splitting up map information, at the same time it is storing the x and z position. So on the fill stage, it places the randomly picked assets in that position. This is still a prototype, I believe there still more work to be done.
