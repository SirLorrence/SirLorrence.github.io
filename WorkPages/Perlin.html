Perlin Noise Experiment

  • Engine: Unity
  • Language: C#

This project I've created an grid with an cube prefab and applied unity's built method of Perlin Noise. But after I became curious tried to learn how the original algorithm works.  

This was difficult to learn(I'm still trying to completely understand it), I created a 2D texture to show me visually how to debug it, then i created my on mesh render with back-face culling  to save on unnecessary  triangles being rendered. 

Made with Bootstrap Studio
